## 安裝與執行步驟 Installation and Execution 
- 將專案複製到本地 → 打開終端機，輸入：
  ```
  git clone -b feature/restaurant_list --single-branch https://github.com/mmlee0819/AC_exercises.git
  ```
- 在本地透過終端機進入資料夾：
  ```
  cd AC_exercises
  cd ac_exercise_restaurantList
  ```
- 於終端機輸入指令：
  ```
  npm install
  npm run dev
  ```
- 若看見以下訊息則代表已順利運行
  ```
  Express is listening on localhost:3000
  ```
- 打開瀏覽器進入以下網址：
  ```
  http://localhost:3000
  ```
- 若要暫停使用，請於終端機按下
  ```
  ctrl + C  
  ```
